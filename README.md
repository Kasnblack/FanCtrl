# FanControl

FanControl은 PC에서 자동으로 팬 속도를 제어할 수 있는 프로그램 입니다.
<br>닷넷 프레임워크 4.6 이상에서 정상 동작합니다.

## 사용한 라이브러리
센서 및 팬 제어 : [LibreHardwareMonitorLib][0]
<br>NZXT Kraken 제어 : [NZXTSharp][1]
<br>USB 통신 : [HidLibrary][2]
<br>Json : [Newtonsoft Json][3]
<br>Graph : [ZedGraph][4]
